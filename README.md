# findmybus
Group Project

Setup dB  
1. Install and start WAMP   
2. Copy data folder, create.txt and load_data.txt  
3. In MYSQL console,  
  
   #create database and tables  
   mysql> source c:\create.txt;  
     
   #load data to tables  
   mysql> source c:\load_data.txt;  
  
Setup Web  
1. Copy findmybus folder to local \wamp\www\*
